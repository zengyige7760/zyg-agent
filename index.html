🛠 技术架构

简洁而强大的技术选型

整个项目仅依赖两个核心第三方库，保持极致轻量的同时，提供完整功能体验。

Python 3.8+   核心语言
Flask   Web 服务框架
SQLite   内嵌数据库
Requests   HTTP 客户端
SSE   实时日志推送
PyArmor   代码混淆保护
📵

单文件部署

整个后端仅由 5 个 Python 文件组成，前端资源内嵌。无需 Node.js、无需 npm build，直接 python app.py 一键启动。真正的零配置开箱即用。

🔐

代码混淆保护

内置 PyArmor 混淆方案，支持 Windows/Linux 跨平台发布。混淆后的代码以 .pyd / .so 格式运行，有效保护源代码知识产权，适合私有化部署和分发。

⚡ 快速开始

三分钟启动你的 AI Agent

从安装到运行，只需三个简单步骤，无需复杂配置。

1

安装依赖

确保已安装 Python 3.8 或更高版本，然后安装所需的三个依赖包。

pip install flask requests flask-cors
2

启动服务

运行启动脚本或直接执行主程序，服务将在 5000 端口启动。

python app.py # Windows 双击 start.bat 亦可
3

配置 API Key

打开浏览器访问管理界面，在「API 提供商」页面设置你的 API Key 并激活。

# 浏览器打开 http://localhost:5000
